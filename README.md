## 特性

- 支持全屏，避免干扰（F11或中键单击托盘图标或从右键菜单中进入）

- 打字机模式（光标固定在屏幕中央）

- 导出 TXT 至程序所在目录

- 支持自定义字体、大小、背景颜色。

- **仅支持纯文本**
